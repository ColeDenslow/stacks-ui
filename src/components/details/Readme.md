## Purpose
Used to display a title and description for something like a product.

## Examples
### Basic use
```jsx
<Details
  title="I'm a title"
  description="I'm a description"
/>
```

### Text alignment
```jsx
<Details
  title="I'm a title"
  description="Left-aligned text (default)"
/>
<Details
  title="I'm a title"
  description="Center-aligned text"
  textAlign="center"
/>
<Details
  title="I'm a title"
  description="Right-aligned text"
  textAlign="right"
/>
```
